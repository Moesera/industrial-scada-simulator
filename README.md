# Industrial SCADA Simulator

This project is a conceptual simulation of an industrial control system, inspired by real-world PLC and SCADA architectures.

It demonstrates:
- Process simulation (temperature control)
- Setpoint-based regulation
- Alarm handling and event logging
- Separation between control logic (PLC) and visualization (SCADA)

The goal is to model how industrial systems behave — not to replicate real hardware.

## Getting Started

Install packages and build node_modules:
```bash
npm i
```

Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
